# Mini Instagram Clone
This project is a simple mini version of Instagram built as a part of a technical task.
It covers basic backend and frontend concepts like authentication, posting, following users and showing a feed.

# Features Implemented
1) User Authentication
1 User signup
2 User login   
3 Password hashing using bcrypt   
4 JWT token based authentication

3) Follow System
   User can follow another user
   User can unfollow a user
   Followers and following count is maintained

4) Posts
   Authenticated users can create posts
   Each post contains image URL and caption
   Users can like a post
   Users can unlike a post

5) Comments
   Users can comment on posts
   Comment shows who commented and the text

6) Feed
   Logged-in user can see feed
   Feed contains posts only from users they follow
   Latest posts appear first

# Tech Stack Used
1) Backend
   Node.js
   Express.js
   MongoDB
   Mongoose
   JWT
   bcryptjs

2) Frontend
   React.js
   Fetch API
   Basic CSS
